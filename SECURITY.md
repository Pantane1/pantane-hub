# Security Policy

## Supported Versions

The latest version of Pantane Hub is actively supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅         |
| Older Versions | ❌         |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Do not disclose vulnerabilities publicly until they have been reviewed and addressed.

You can report security issues by opening a private security advisory on GitHub or by emailing me directly @ pantane254gmail.com.

Please include:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Screenshots or proof of concept if applicable

We will acknowledge reports as soon as possible and work to resolve confirmed issues promptly.

Thank you for helping keep Pantane Hub secure.
